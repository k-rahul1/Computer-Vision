****Calibration and Augmented Reality****


Operating System - Ubuntu 20.04.5 LTS
IDE - VS Code

Following folders are available:

1. calibration
---contents: calibration.cpp, Makefile, function.cpp, function.h
Instructions to run the program:
1. Type 'make' in the terminal to build the file.
2. Type './calib' to run the executable
Note: This program requires chessboard in the image and also video is streamed through mobile (Change the ip before running)

2. augmentedReality
---contents: augmentedReality.cpp, Makefile, samsung_intrinsic_parameters.xml
Instructions to run the program:
1. Type 'make' in the terminal to build the file.
2. Type './calib' to run the executable
Note: This program requires chessboard in the image and also video is streamed through mobile (Change the ip before running)

3. ORB
---contents: feature.cpp, Makefile
Instructions to run the program:
1. Type 'make' in the terminal to build the file.
2. Type './orb' to run the executable
Note: Video is streamed through mobile (Change the ip before running)

4. Extension
---contains folders: face.cpp, Makefile, haarcascade_frontalface_default.xml, haarcascade_mcs_eyepair_big.xml, Sunglasses1.png
Instructions to run the program:
1. Type 'make' in the terminal to build the file.
2. Type './face' to run the executable

5. Project4_Report.pdf: Report of project

6. readme.txt: instructions to rum the program
